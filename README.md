# Emma Portfolio

Palantir-inspired portfolio built with Next.js, Tailwind, and GSAP.

## Scripts

```bash
npm run dev
npm run lint
npm run build
npm run start
```

## Update Content

- Edit the copy, projects, and links in `src/app/page.tsx`.
- Replace contact links with real profiles.

## Notes

- Uses Space Grotesk and IBM Plex Sans via `next/font`.
- GSAP handles intro and scroll reveal animations.
